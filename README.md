# MonkeyBox

## 简介

一个 Out of Box 的猴子脚本框架。

## Usage

```JavaScript
// @grant GM_setValue
// @grant GM_getValue
// @grant GM_deleteValue
// @grant GM_listValues
// @grant GM_addValueChangeListener
// @grant GM_removeValueChangeListener
// @require https://github.com/bramblex/MonkeyBox/raw/master/monkey-box.user.js
```

详见 `src/dev-page`
